# MakeItViral 🚧 (Work in Progress)
An AI system that gives Instagram creators viral reel ideas based on their niche and past posts. 

*Note: This project is currently halfway done and still in active development!*

### 🛠️ Tech Stack
* Python, AI

### ✨ Key Features
* **Custom Idea Generator:** Suggests new reel ideas that match the creator's specific style.
* **Real Examples:** Shows actual Instagram reels as inspiration for the ideas.
* **Virality Rating:** Scores a reel idea to predict how likely it is to go viral.
